# Krypto-Trading Dashboard

Ein automatisiertes Krypto-Trading Dashboard zur Verwaltung von Trading-Bots, Wallet und Performance-Tracking.

## Features

- **Dashboard**: Übersicht über aktuelle Trades und Performance
- **Trading-Bots**: Automatisierte Krypto-Trading Bots erstellen und verwalten
- **Wallet**: Einzahlungen, Auszahlungen und Transaktionshistorie
- **KYC-Verifizierung**: Identitätsverifizierung für sichere Transaktionen
- **Support-System**: Ticket-System für Kundenanfragen
- **Admin-Panel**: Verwaltung von Nutzern, Bots und Transaktionen

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (Auth, Database, Storage, Edge Functions)

## Installation

```bash
# Dependencies installieren
npm install

# Development Server starten
npm run dev
```

## Deployment

```bash
# Production Build erstellen
npm run build

# Build lokal testen
npm run preview
```

## Umgebungsvariablen

Erstelle eine `.env` Datei mit folgenden Variablen:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
